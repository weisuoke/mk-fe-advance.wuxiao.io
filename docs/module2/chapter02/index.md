看起来是基础知识，但是有很多难点

### 将收获什么

- TypeScript
  - 为什么？
  - 基础知识点
  - interface
  - 泛型
  - 高级特性
  - 定义文件
- vue3
  - 为什么？
  - 基础知识点
  - 响应式对象基本原理的解析
  - 副作用
  - 自定义 Hooks
- Vue3 和 React 的异同

- 泛型
