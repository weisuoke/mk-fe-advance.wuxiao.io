## 快速入门第一个脚手架

创建一个文件夹

```shell
mkdir imooc-test
```

初始化项目

```shell
npm init -y
```

新建一个`bin`目录

```shell
mkdir bin
```

在`bin`目录下新建index.js

```js
// bin/index.js
#!/usr/bin/env node

console.log('welcome imooc-test');
```

去修改package.json中的内容

```js
// package.json

{
  // ...
  "bin": {
    "imooc-test": "bin/index.js"
  }
  //...
}
```

去发布cli

```shell
npm login
npm publish
```

发布后打开终端

```
npm i -g imooc-test
```
